Message Posting

You can use the commands 't!message' and 't!embed' to get the bot to send a message or embeded message.

  • t!message content : Sends a message in the channel that the command was executed with the content of the rest of the command.
  • t!embed json : Sends an embeded message in the channel that the command was executed with the content from the json of the command. Click here to go to the embed json generator.

Basic Moderation

You can change the bot configs for the server and ban/kick users.

  • t!config setting parameters : Set the specified setting to the specified parameter.
  • t!ban user reason : Bans the specified user from the current server with the specified reason. The user can be specified by their id or by mentioning them.
  • t!kick user reason : Kicks the specified user from the current server with the specified reason. The user can be specified by their id or by mentioning them.

User & Server Info

Get information about a user or a server including their icon, id, and creation date.

  • t!userinfo user : Gets the information about a user from the current server. The user can be specified by their id or by mentioning them.
  • t!guildinfo : Gets the information about current server.
