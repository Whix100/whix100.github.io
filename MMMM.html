Minecraft Mass Model Maker

Minecraft Mass Model Maker (MMMM) is a simple program designed to help mod makers create default model files for multiple items and blocks at a time. You can input items and blocks one at a time or import a list from a text file. You can also choose the modid to use, and where you want the model files to be outputted to.
